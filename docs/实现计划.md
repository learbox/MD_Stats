#### 1.2 重构 `__init__`（动态行创建）

当前 `__init__` 硬编码 8 行。改为接收可选的 `rows` 参数：

```python
def __init__(self, parent=None, rows=None):
    ...
    self._rows: tuple[str, ...] = tuple(rows) if rows else _DEFAULT_ROWS
    ...
    # 遍历 self._rows 创建标签
    for r, text in enumerate(self._rows):
        lbl = QLabel(text)
        ...
        val = QLabel("-")
        ...
```

`setFixedSize` 的高度需要根据行数动态计算。目前的 `_DEFAULT_H = 330` 对应 8 行。可以按每行约 35px 估算：`_DEFAULT_H = 20 + 20 + len(rows) * (20 + 6) — 6`（上下边距 + 行高累加）。

更准确的做法：保持一个基准行高，`_DEFAULT_H = 20 * 2 + len(rows) * 26`（上下边距 20，每行含间距约 26px）。

#### 1.3 新增 `set_rows` 方法

用于运行时通过 `update_style` 动态重建行（当 config 中的 `rows` 变了）：

```python
def set_rows(self, rows: list[str]) -> None:
    """动态替换显示行。清空现有标签后重建。"""
    new_rows = tuple(rows) if rows else _DEFAULT_ROWS
    if new_rows == self._rows:
        return  # 无变化，跳过
    self._rows = new_rows
    # 清空旧控件
    for lbl in self._labels + self._values:
        self._grid.removeWidget(lbl)
        lbl.deleteLater()
    self._labels.clear()
    self._values.clear()
    # 重建
    for r, text in enumerate(self._rows):
        lbl = QLabel(text)
        lbl.setAlignment(...)
        self._grid.addWidget(lbl, r, 0)
        self._labels.append(lbl)
        val = QLabel("-")
        val.setAlignment(...)
        self._grid.addWidget(val, r, 1)
        self._values.append(val)
    self._apply_style()
    # 调整窗口高度
    h = 40 + len(self._rows) * 26
    self.setFixedSize(self.width(), h)
```

#### 1.4 重构 `update_content`

不再用固定索引。改为遍历 `self._rows` 查 `_ROW_KEY_MAP`：

```python
def update_content(self, deck_name: str, stats: dict | None) -> None:
    if stats is None:
        for v in self._values:
            v.setText("-")
        return

    for i, row_name in enumerate(self._rows):
        keys = _ROW_KEY_MAP.get(row_name)
        if keys is None:
            self._values[i].setText("-")
            continue

        if len(keys) == 1:
            # 单值行
            key = keys[0]
            if key == "卡组":
                self._values[i].setText(deck_name or "(未指定)")
            else:
                self._values[i].setText(str(stats.get(key, "-")))
        else:
            # 合并行 (len == 2): 显示 "v1 / v2"
            v1 = stats.get(keys[0], 0)
            v2 = stats.get(keys[1], 0)
            self._values[i].setText(f"{v1} / {v2}")
```

新增 `_bg_pixmap` 属性：

```python
def __init__(self, ...):
    ...
```

```python
    """按 config.toml [floating_window] 段更新外观。
    float_bg_path: theme.toml 中的 float_bg 图片路径（可选）。
    """
    ...
    # ---- 处理背景图 ----
    if float_bg_path:
        pm = QPixmap(float_bg_path)
        if not pm.isNull():
            self._bg_pixmap = pm
        else:
            self._bg_pixmap = None
    else:
        self._bg_pixmap = None
    ...
```

修改 `paintEvent`：

```python
def paintEvent(self, event) -> None:
    painter = QPainter(self)
    painter.setRenderHint(QPainter.RenderHint.Antialiasing)
    path = QPainterPath()
    margin = 8
    path.addRoundedRect(margin, margin,
                        self.width() - margin * 2,
                        self.height() - margin * 2, 10, 10)

    if self._bg_pixmap is not None:
        # 有背景图：先用纯色打底，再叠加背景图
        painter.fillPath(path, self._bg_color)
        # 在圆角矩形区域内绘制背景图
        painter.setClipPath(path)
        scaled = self._bg_pixmap.scaled(
            self.size(),
            Qt.AspectRatioMode.IgnoreAspectRatio,
            Qt.TransformationMode.SmoothTransformation,
        )
        painter.drawPixmap(0, 0, scaled)
        painter.setClipping(False)
    else:
        # 无背景图：纯色填充（兜底逻辑）
        painter.fillPath(path, self._bg_color)
    painter.end()
```

这就是兜底逻辑的核心：`if self._bg_pixmap is not None` 走图片路径，`else` 走纯色路径。图片加载失败（`isNull()`）和未配置图片时，`_bg_pixmap` 都是 `None`，自然回退到纯色。

### 第五步：修改 `ui/main_window.py`（三处改动）

#### 5.1 重写 `_copy_to_clipboard`

```python
def _copy_to_clipboard(self) -> None:
    """将统计表格复制到系统剪贴板。

    行为受 config.toml [clipboard] 段控制：
      - vertical_layout: true=竖排(key\\tvalue)，false=横排 TSV
      - scope: "current"=当前输入框中卡组，"all"=全部卡组含合计
      - columns: 要复制的列名列表，空=全部 16 列
    """
    from src.config import load_config
    cfg = load_config().get("clipboard", {})

    records = load_records()
    stats = compute_stats(records)

    columns = cfg.get("columns") or STATS_COLUMNS  # 空列表=全选
    scope = cfg.get("scope", "all")
    vertical = cfg.get("vertical_layout", False)

    if scope == "current":
        deck_name = self._deck_input.text().strip()
        stats = [s for s in stats if s.get("卡组") == deck_name]
        if not stats:
            stats = [s for s in stats if s.get("卡组") == "合计"]  # 兜底

    if vertical:
        # 竖排模式
        lines: list[str] = []
        for row in stats:
            deck = row.get("卡组", "")
            if scope == "all" or len(stats) > 1:
                lines.append(f"[{deck}]")
            for col in columns:
                if col == "卡组":
                    val = deck
                else:
                    val = row.get(col, "")
                lines.append(f"{col}\t{val}")
        QApplication.clipboard().setText("\n".join(lines))
    else:
        # 横排模式（传统 TSV）
        lines = ["\t".join(columns)]
        for row in stats:
            lines.append("\t".join(str(row.get(c, "")) for c in columns))
        QApplication.clipboard().setText("\n".join(lines))

    self._show_status("已复制统计表格到剪贴板")
```

### 潜在风险与注意事项

| 风险点 | 说明 | 缓解措施 |
|--------|------|----------|
| 行数变化导致窗口高度不足 | 用户配置少于默认 8 行时窗口偏大，多于 8 行时溢出 | `set_rows` 时动态调整 `setFixedSize` |
| `_ROW_KEY_MAP` 未覆盖用户输入的行名 | 用户拼写错误导致显示 `-` | `update_content` 中 keys 为 None 时回退到 `"-"` |
| `config.toml` 缺少新段时程序崩溃 | `dict.get()` 访问缺少 key 的配置 | 所有读取使用 `.get(key, default)` 带默认值 |
| `float_bg` 图片路径在 Windows 下的反斜杠问题 | QPixmap 加载路径 | `_collect_image_paths` 已用 `as_posix()` 统一处理，无需额外改动 |
| 主题切换后悬浮窗背景图未更新 | `_on_reload_config` 中需要同时传新路径 | 已在第五步 `_on_reload_config` 中处理 |
