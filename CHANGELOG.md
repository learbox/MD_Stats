# 更新日志

## v1.6.0 (2026-06-09)

### 新功能
- **段位升降检测**：硬币阶段同步检测升段/降段局，CSV 新增"段位升降"列
- **段位胜率统计**：新增升段/降段次数和对应胜率
- **统计表格列选择**：设置中自定义显示哪些统计列
- **系统气泡通知**：对局结束时弹出通知，可配置时长
- **系统托盘**：最小化到托盘 + 右键菜单
- **系统标签页**：整合日志模式、通知、托盘配置

### 改进
- `config.toml` 丢失时自动生成默认配置
- 截图文件名含游戏分辨率
- CSV 改为原子写入，防止数据丢失
- `.app_state.json` 列宽保存跳过隐藏/拉伸列

### 修复
- BitBlt 截图异常不再中止工作线程
- DualListWidget 嵌套替换失效导致设置不生效
- 启动时窗口最小化误判为就绪
- 缺少 rank 模板时检测无法启动
- 托盘图标不显示
